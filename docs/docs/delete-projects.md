# How to Delete a Project

A project **can only be deleted on the mobile app** and only **if it has not been previously uploaded**. If a project has been uploaded at least once and you want to remove it from the active projects list, you can [archive it](https://help.docusketch.com/docs/archiving-projects) (opens in new tab).

On the mobile app, find the project you want to delete, tap the **three dots** on its thumbnail, and select "**Delete**":

![Find the project, tap its 3-dot menu, and choose Delete](../assets/delete-projects/01-delete.png)
