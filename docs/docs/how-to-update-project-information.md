# How to Update Project Information

## On Mobile App

Tap the **project thumbnail**, then tap the **project information** to update its details.

![Tap the project thumbnail, then tap its project information](../assets/how-to-update-project-information/01-open.png)

All additional project details can also be edited from here:

![The edit screen with the project name, address, and loss details](../assets/how-to-update-project-information/02-edit.png)

Once done, tap the **back** button and go back to the project list, then tap the **three dots** on your project and select "**Upload**":

![Go back to the project list, open the 3-dot menu and tap Upload](../assets/how-to-update-project-information/03-upload.png)

## On Web Portal

Locate the project you want to edit, click on the "**More**" button and select "**Edit project info**":

![On the Web Portal, click More and choose Edit project info](../assets/how-to-update-project-information/04-web-edit.png)

Update the information and click "**Save**" when done:

![Update the information and click Save](../assets/how-to-update-project-information/05-web-save.png)
