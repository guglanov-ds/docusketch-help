# How to Add a Timeline

Find the initial 360° tour (not a timeline) in your project list and tap the
**3 dots**. Select "**Add timeline**." Fill out the timeline information and tap
"**Next**":

![Open the project's 3-dot menu, choose Add timeline, then fill in the timeline information and tap Next](../assets/how-to-add-a-timeline/01-add-timeline.png)

If you want to view your previous 360° image, tap **the 360° image preview** for
the room. To return, tap the **cross icon**:

![Tap a room's 360° image preview to view the previous image; tap the cross icon to return](../assets/how-to-add-a-timeline/02-view-360.png)

To start capturing a new 360° image for the room, tap "**Take new 360° image**"
and complete the shooting process. Once the 360° image is taken, you will see a
green check icon in the upper-left corner of the room preview:

![Tap Take new 360° image and complete shooting; a green check appears on the room once captured](../assets/how-to-add-a-timeline/03-take-360.png)

The three dots icon next to the "Take new 360° image" button can be tapped to
either **open the 360° image** or **remove it from the timeline**:

![The three-dot menu next to Take new 360° image lets you open the 360° image or remove the room](../assets/how-to-add-a-timeline/04-room-menu.png)

After you updated all the 360° images you needed to, tap "**Done**" and return to
the project list by tapping **the back arrow**:

![Tap Done once all rooms are captured, then the back arrow to return to the project list](../assets/how-to-add-a-timeline/05-done.png)

**Upload** the timeline:

![Open the timeline's 3-dot menu and tap Upload](../assets/how-to-add-a-timeline/06-upload.png)

!!! note
    If you try to upload a timeline for the initial 360° tour (parent tour) that
    hasn't been uploaded yet, a warning message will prompt you to upload the
    initial 360° tour first.

![The "Initial 360° upload is required" warning shown when the parent tour hasn't been uploaded](../assets/how-to-add-a-timeline/07-parent-warning.png)
